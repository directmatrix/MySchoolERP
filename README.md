# MySchoolERP
School Management Solution that works Offline ,Online &amp; Mobile App. 
## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and trial purposes. 

### Prerequisites

.NET Framework
SQL Server 2012
Android 6.0 Upwards
Crystal Reports
